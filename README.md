<p align="center">
PCShop is a computer hardware store web application developed as a personal project.
<br><br>
This project is designed as a <b>long-term evolving system</b>, where I continuously add new features, improvements, and architectural refinements as I expand my knowledge and development skills.
<br><br>
The goal is to simulate a real-world e-commerce system focused on computer hardware while practicing backend development, database design, and web application structure.
</p>
